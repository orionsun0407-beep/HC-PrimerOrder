# HC PrimerOrder

Public static version of the SnapGene primer order helper.

## Use

Open `index.html`, upload:

1. A SnapGene-exported primer text file.
2. A `.xlsx` primer order template.

The workbook is edited in the browser and downloaded with today's date in the file name.
The order template can be saved in the browser with a custom template name and reused later.

## Privacy

All processing happens in the visitor's browser. Primer files and order templates are not uploaded to a server by this static version.

## Deploy

This folder is ready for GitHub Pages, Netlify, Vercel static hosting, or any static file server.
